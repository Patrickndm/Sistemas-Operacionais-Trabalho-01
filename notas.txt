Etapas para o push na branch remota main:
1) Modifico os códigos na branch local patrick
2) Checo os arquivos modificados usando status
3) Faço add e commit da branch local patrick
4) Dou um push da branch local patrick para a branch remota patrick (por segurança)
5) Mudo para a branch local main
6) Atualizo a branch local main com o pull da branch remota main
7) Faço merge da branch local main com a branch local patrick
8) Dou push da branch local main para a branch remota main

Etapas para o merge na branch local patrick:
1) Mudo para a branch local main
2) Dou pull da branch remota main
3) Mudo para a branch local patrick
4) Dou merge da branch local main

Para ver todas as branchs (remotas ou locais):
git branch -a
